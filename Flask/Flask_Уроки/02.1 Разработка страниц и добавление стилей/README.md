## Последовательность выполнения работы  
1. Создали виртуальную среду venv через консоль  
2. Зашли в виртуальную среду  
3. Установили Flask через консоль pip install flask  
4. Установили библиотеку, для работы с базами данных (вместо SQL-запросов используем Python-классы).через консоль pip install flask-sqlalchemy  
5. Создали файл app.py в корне проекта. Главный файл (Controller + настройки Flask)  
6. Запустили файл app.py  
7. Перешли в браузере по адресам (проверили работу приложения):  
  - http://127.0.0.1:5000  
  - http://127.0.0.1:5000/one  
  - http://127.0.0.1:5000/home  
  - http://127.0.0.1:5000/user/alex/123  
8. Создали папку templates в корне проекта, в ней хранятся HTML-шаблоны (View в MVC)  
9. Создали общий шаблон base.html в папке  templates.  
10. Для быстрого заполнения базовой структуры html, в VSCode или PyCharm ныбираем «!» и Tab  
11. Создали шаблоны наследники, файл index.html  и one.html в папке  templates   
12. Создали папку static. Статические файлы в папке такие как, CSS/JS/Картинки.  
13. Создали папку css в папке static  
14. Создали файл стилей main.css в папке css  
15. Подключаем стиль к общему шаблону  base.html через функцию url_for()  
16. Перешли в браузере по адресам (проверили работу приложения):  
  - http://127.0.0.1:5000  
  - http://127.0.0.1:5000/one  
  - http://127.0.0.1:5000/home
