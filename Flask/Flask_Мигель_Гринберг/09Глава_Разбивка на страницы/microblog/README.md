<h1 align="center" id="title1">Глава 9: Разбивка на страницы</h1>

Файловая структура:  

```
microblog/
│
├── app/						# Главный пакет приложения
│	├── templates/				# Папка с HTML шаблонами
│	│	├── _post.html			# Вложенный шаблон постов для использования в user.html и index.html 
│	│	├── 404.html
│	│	├── 500.html
│	│	├── base.html			# Базовый шаблон
│	│	├── edit_profile.html	# Шаблон формы для редактирования профиля пользователя.
│	│	├── index.html			# Шаблон главной страницы
│	│	├── login.html			# Шаблон формы для входа в систему
│	│	├── register.html		# Шаблон формы для регистрации в системе
│	│	└── user.html			# Шаблон страницы профиля пользователя
│	├── __init__.py				# Подкаталог, содержащий __init__.py файл, считается пакетом и может быть импортирован.
│	├── errors.py
│	├── forms.py				# Модуль для хранения классов веб-форм
│	├── models.py				# Модель пользовательской базы данных
│	└── routes.py				# Маршруты на страницы. Функции просмотра.
├── migrations/					# Репозиторий миграции БД
│	├── versions/				# Папка с версиями миграции
│	└── ...
├── logs/
├── .flaskenv					# Переменные среды для команды flask, находит файл запуска
├── app.db						# База данных
├── config.py					# Основной конфигурационный файл
├── microblog.py				# Основной модуль приложения (запуск приложения)
├── requirements.txt			# Список зависимостей
└── tests.py					# Модульное тестирование
```
